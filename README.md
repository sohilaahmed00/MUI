# Vortex - Modern Next-Generation UI

![Vortex Preview](./public/Screenshot%202026-04-21%20033735.png)

A breathtaking, premium web application user interface built using **React** and **Material-UI (MUI)**. This project demonstrates advanced component styling, responsive web design, and a highly polished custom "Dark Mode" aesthetic.

---

## ✨ Features

- **Premium Custom Theme**: A meticulously crafted cohesive palette featuring Slate Blues (`#2C3947`), elegant Sand/Gold accents (`#C2A56D`), and deep navy backgrounds.
- **Glassmorphism & Depth**: Extensive usage of UI trends like `backdropFilter: blur(20px)`, subtle glow effects, and radial gradients to create floating depth.
- **Responsive Grids & Flexbox**: 
  - Symmetrical `1:1` aspect ratio feature cards perfectly aligned using MUI Grids.
  - A flawless, side-by-side authentication form that gracefully adapts to mobile devices using Flex layouts.
- **Custom Components**: Clean, borderless Navbar with smooth scrolling, tailored Button overrides without messy artifacts, and a fully distributed Footer.

## 🛠️ Technology Stack
- **Frontend**: React.js
- **UI Library**: Material-UI (MUI) v5
- **Styling**: `sx` prop styling, `createTheme` overrides, and vanilla CSS for scrollbars/selections.
- **Routing**: React Router DOM

---

## 🚀 Getting Started

To run this project locally:

1. Clone the repository.
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm start
   ```
